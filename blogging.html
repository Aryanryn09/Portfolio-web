Blogging website

The blog application is a CRUD (Create, Read, Update, and Delete) web application that allows users to post new blogs, view existing blogs, update existing blogs, and delete blogs. The application is built using the Express.js framework, which is a popular framework for building web applications in Node.js. It uses MongoDB as the database to store blog data.

The application has several routes that allow the user to perform CRUD operations. The "create" route is used to add a new blog post to the database. To create a new blog post, the user can fill out a form that includes fields for the blog title, author name, and content. Once the form is submitted, the data is sent to the server using the HTTP POST method. The server then adds the new blog post to the MongoDB database.

To update an existing blog post, the user can navigate to the "update" route, which retrieves the blog post data from the database and pre-populates a form with the current values. The user can then update the form and submit the changes using the HTTP PUT method, and the server updates the blog post in the database accordingly. To delete a blog post, the user can navigate to the "delete" route and specify the blog post to delete. link to project
